Intro


이번 포트폴리오 화면을 1학기 첫 시험때 했던 고양이 웹 페이지의 연장선으로 직접 그린 일러스트 고양이로 정했습니다.
고양이는 나와 가장 닮은 동물이며 내가 가장 사랑하는 동물이기에 선정하였습니다.
고양이와 닮은 나는 계원예술대학교 2학년 재학중인 C반의 조예니입니다.


● 웹 퍼블리싱을 하면서 나만의 페이지를 만드는 법을 배워갔고 그렇게 우리 고양이를 소개하는 페이지를 만들 수 있었다. 우리집 고양이와 가족을 소개하는 페이지와 좋아하는 간식을 별점으로 표시하는 페이지를 만들 수 있었다.


● 우리 고양이의 하루일상과 만난 날부터 시작하여 지금까지의 일을 년도 순으로 나타내어 표현했습니다.

웹 표준 퍼블리싱


● 웹 표준 퍼블리싱이란 특정 브라우저에서만 사용되는 비 표준화된 기술은 배제하고, 모든 브라우저에 동일하게 적용될 수 있도록 W3C(World Wide Web Consortium)의 토론을 통해 나온 권고안을 사용하는 것을 말한다. 웹 문서의 구조와 표현, 그리고 동작을 구분해서 사용하는 것을 뜻한다.

웹 접근성


● 신체적 제한조건을 가진 고령자나 장애인들도 이용하기에 편리하도록 웹을 구성, 디자인 하는 것이다.

반응형 웹


● 다양한 해상도에 대응하기 위하여 디바이스의 종류에 따라 웹 페이지 크기가 자동적으로 재조정 되는 것을 말한다. 어떤 환경에서도 그에 맞게 사이즈가 변화되어 사용자가 보기 편리하게 만드는 웹이다. ‘모바일웹+PC웹’과의 차이점은 하나의 HTML만으로도 특정장치에서 최적화 된 환경을 제공한다는 것이다.
